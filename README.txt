# GreenLiving Store (Demo E-commerce Website)

This folder contains a 6-page front-end website for coursework.

## Pages
- `index.html` — Home (includes brief descriptions of other pages + dynamic featured products)
- `products.html` — Product listings (name, actual price, discounted price, description, tags, table)
- `blog.html` — Blog demonstration (technology impact on daily life & business)
- `research.html` — Research comparison table + 5 reference sites
- `about.html` — About us portfolios for team members
- `contact.html` — Functional form with JavaScript validation

## Folder Structure
- `css/main.css` — external CSS (site-wide)
- `js/main.js` — dynamic content, theme toggle, cart badge
- `js/form.js` — form validation logic
- `images/` — logo + placeholder product images (SVG files saved with .jpg names)

## How to run
Open `index.html` in a browser, then navigate using the navbar.

> Tip: For best results, run via a simple local server (optional), e.g., VS Code Live Server.
